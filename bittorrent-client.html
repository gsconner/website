Bittorrent Client

GitHub

A Python BitTorrent client designed to adhere to BitTorrent protocol specification.

Features communication with BitTorrent trackers, and downloading / uploading data with other peers.

Fully downloads torrents using a .torrent metafile.

Acquires peer via communication with tracker through UDP, concurrently downloads and uploads data to peers over TCP.

Collaborated in a team of four, solely responsible for implementing TCP communication and managing data exchange between clients
